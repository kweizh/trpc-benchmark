### Key tRPC v11 details

- **`t.createCallerFactory(appRouter)`** — the v11 API for direct server-side invocation. The returned factory is called once per request: `createCaller({})` (context object can be extended with auth/DB handles).
- **Zod v4 input validation** — `z.object({ text: z.string().min(1) })` rejects empty strings at the tRPC layer before the handler runs.
- The mutation returns `{ success: true as const, message: string }` — the literal type is preserved through the Server Action's return signature so the client is fully type-safe.
