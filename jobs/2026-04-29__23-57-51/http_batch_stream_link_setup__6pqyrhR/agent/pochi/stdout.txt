## How It Works
1. Client sends a GET request with header `trpc-accept: application/jsonl`
2. Server detects the streaming accept header and responds with JSONL-formatted chunks
3. Each chunk is transmitted as it's yielded: `"Hello"` → `", "` → `"World"` → `"!"`
4. `@trpc/react-query` progressively updates `query.data` as `["Hello"]` → `["Hello", ", "]` → `["Hello", ", ", "World"]` → `["Hello", ", ", "World", "!"]`
5. The component re-renders with each update, displaying the text growing until the final `Hello, World!`
