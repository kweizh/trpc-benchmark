### 2. Client Configuration Update (client.ts)
The key breaking change in tRPC v11 is that the transformer configuration must be moved from the top-level client options into the link configuration:

**Before (v10):**
```typescript
const client = createTRPCProxyClient<AppRouter>({
  transformer: superjson,  // ← Top-level
  links: [
    httpBatchLink({
      url: 'http://localhost:3000',
    }),
  ],
});
```

**After (v11):**
```typescript
const client = createTRPCProxyClient<AppRouter>({
  links: [
    httpBatchLink({
      url: 'http://localhost:3000',
      transformer: superjson,  // ← Moved into httpBatchLink
    }),
  ],
});
```
