## Changes

**`server.ts`** — added `errorFormatter` to `initTRPC.create()`:

```ts
export const t = initTRPC.create({
  errorFormatter(opts) {
    const { shape, error } = opts;
    return {
      ...shape,
      data: {
        ...shape.data,
        zodError:
          error.code === 'BAD_REQUEST' && error.cause instanceof ZodError
            ? error.cause.flatten()
            : null,
      },
    };
  },
});
```

Also addressed two pre-existing issues that prevented `npm start` from working so the change could be verified:
1. The `app.use(express.json(), '/trpc', trpcExpress.createExpressMiddleware(...))` call had middleware before the path — split into two `app.use` calls.
2. There was no `tsconfig.json`, which caused ts-node to fail silently against TypeScript 6.0's deprecated module-resolution defaults. Added a minimal `tsconfig.json`.
