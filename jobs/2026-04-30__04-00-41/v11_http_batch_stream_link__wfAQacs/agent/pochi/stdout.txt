## `server.ts`
Uses `initTRPC.create()`, defines a `chatStream` query as an `async function*` yielding `"hello"`, `"world"`, `"!"`, and serves it with `createHTTPServer` from `@trpc/server/adapters/standalone` on port 3000.

## `client.ts`
Uses `createTRPCClient` configured with `httpBatchStreamLink({ url: 'http://localhost:3000' })`. It calls `trpc.chatStream.query()` which returns an async iterable, then iterates with `for await` and logs each chunk.
