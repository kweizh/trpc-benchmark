### How streaming works
1. **Server** — `streamWords` is a `publicProcedure.query(async function* () {...})` that yields `"Hello"` → `"streaming"` → `"world"` with 100ms delays.
2. **Transport** — `httpBatchStreamLink` sends `trpc-accept: application/jsonl`; the server responds with newline-delimited JSON (JSONL), flushing each word as it's yielded.
3. **Client** — The vanilla tRPC client's `.query()` returns `Promise<AsyncIterable<string>>` for generator procedures. The page component consumes it with `for await...of` in a `useEffect`, appending each word to state as it arrives.
